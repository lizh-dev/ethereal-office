5. 量産アーキテクチャ

100テンプレートを効率的に作る仕組み。

1

ゾーンジェネレーター (コード)

SpaceWizardの生成ロジックを抽出。各ゾーンタイプの生成関数:

generateDeskArea()
generateMeetingRoom()
generateLounge()
generateCafe()
generateClassroom()
generateCallCenter()
generateReception()
generateLibrary()

白: 既存 / 黄: 新規追加

2

テンプレート定義 (JSONデータ)

各テンプレートは「ゾーンの組合せ」で定義。コードを書かずにデータで量産:

{
  id: "startup-4",
  name: "スタートアップ 4人",
  category: "startup",
  teamSize: "small",
  theme: "standard",       // ← スタイル軸
  zones: [
    { type: "desk-area", rows: 2, cols: 2, x: 50, y: 50 },
    { type: "meeting", seats: 4, x: 300, y: 50 },
    { type: "lounge", x: 300, y: 250 },
  ]
}
// 同じレイアウトをテーマ違いで複製:
{ ...startup4, id: "startup-4-cyber", theme: "cyberpunk" }
{ ...startup4, id: "startup-4-jp",    theme: "japanese" }
3
